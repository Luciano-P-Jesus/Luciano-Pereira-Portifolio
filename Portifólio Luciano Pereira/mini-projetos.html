Space InPython

Jogo estilo Space Invaders feito com Pygame. O jogador controla uma nave espacial, destrói ondas de inimigos e coleta power-ups como escudo, tiro duplo, velocidade extra e vida extra. O jogo possui sistema de níveis progressivos — a cada fase os inimigos ficam mais rápidos e agressivos.

Python Pygame Game Dev
🎮 Controles fluidos com teclado
⚡ 4 tipos de power-ups
📈 Dificuldade progressiva por nível
💥 Sistema de colisão e vidas

Python HTML5 CSS3 JavaScript
🔢 Algoritmo oficial de validação

03

To-Do List

Aplicação web de gerenciamento de tarefas com interface moderna e limpa. Permite adicionar, completar e remover tarefas com persistência via localStorage — suas tarefas continuam salvas mesmo ao fechar o navegador. Inclui contador de tarefas ativas e opção de limpar tudo.

HTML5 CSS3 JavaScript
💾 Dados salvos no localStorage
✅ Marcar como concluída com animação
🧹 Limpar tudo com confirmação

Nomear Tudo

Dois scripts Python utilitários para automação de arquivos. O primeiro renomeia em lote todos os arquivos de uma pasta seguindo um padrão definido pelo usuário. O segundo cria arquivos .txt com nomes aleatórios — perfeito para testes e simulações rápidas.

Python CLI Automação
📁 Renomeação em lote de arquivos
🎲 Criação de arquivos aleatórios
⌨️ Interface via linha de comando
