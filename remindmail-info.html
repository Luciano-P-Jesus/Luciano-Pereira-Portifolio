RemindMail

Aplicação Full Stack Avançada

Sobre o Projeto

RemindMail é uma aplicação web moderna e robusta desenvolvida com React e Vite no frontend, utilizando TypeScript para tipagem estática. O projeto também possui um backend complexo em Node.js com Express, banco de dados PostgreSQL, autenticação JWT e integração com serviços de email.

Tecnologias Utilizadas

Frontend

  • React 19
  • Vite
  • TypeScript
  • Tailwind CSS
  • Framer Motion
  • Axios

Backend

  • Node.js
  • Express 5
  • PostgreSQL
  • JWT (Autenticação)
  • Nodemailer
  • Bcrypt

Características Principais

  • ✓ Sistema de autenticação completo com JWT
  • ✓ Dashboard interativo com Framer Motion
  • ✓ Modal de contato profissional
  • ✓ Integração com serviço de email
  • ✓ Componentes reutilizáveis e bem estruturados
  • ✓ Criptografia de senhas com Bcrypt

⚠️ Sobre o Acesso

Este é um projeto full stack complexo que requer:

  • • Configuração de banco de dados PostgreSQL
  • • Variáveis de ambiente (.env) com credenciais
  • • Backend rodando em servidor separado
  • • Build do frontend React com Vite
